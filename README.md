# Job Portal System

A full-stack job portal application built using MERN stack.

## Features
- User Authentication
- Job Posting
- Job Application
- Recruiter Dashboard

## Tech Stack
- React
- Node.js
- Express
- MongoDB

## How to Run

### Backend
cd backend  
npm install  
npm run dev  

### Frontend
cd frontend  
npm install  
npm run dev
